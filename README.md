# MongoDB-Essential
## Commands
***Start MongoDB***
```console
mongodb --dbpath mongod_only
```
***Open MongoDB shell***
```console
mongosh
```
***Sow default databases***
```console
show databases
```
***Store inquiry data***
```console
db.test.insertOne({"Hello": "World"})
```
